# 🚛 TransitOps: Smart Transport Operations Platform

A centralized, role-based platform designed to digitize transport operations—managing the complete lifecycle of vehicles, drivers, dispatch, maintenance, and operational finance.

## 🎯 Hackathon Objective

Built to replace manual spreadsheets, TransitOps enforces strict business rules (e.g., blocking dispatch for overloaded vehicles or suspended drivers) while providing actionable insights through a clean, responsive web interface.

## 👥 Target Personas & RBAC

1. **Fleet Manager:** Oversees fleet assets, maintenance logs, and operational efficiency.
2. **Driver:** Manages trips and updates active delivery statuses.
3. **Safety Officer:** Monitors driver compliance, license validity, and safety scores.
4. **Financial Analyst:** Reviews fuel consumption, maintenance costs, and Vehicle ROI.

## 🚀 Features & Bonus Deliverables

- **Secure Authentication & RBAC:** Dynamic dashboards based on user login.
- **Real-Time Validation Engine:** Automatic state transitions (e.g., adding a vehicle to maintenance automatically hides it from the dispatch pool).
- **Advanced Data Tables:** Built-in search, filtering, and sorting capabilities.
- **Visual Analytics:** Interactive KPI tracking for Fleet Utilization.
- **UI/UX:** Responsive design with an integrated **Dark Mode** toggle.
- **Reporting:** One-click CSV exports for operational ledgers.

## 💻 Tech Stack

- **Frontend:** React.js, standard CSS3 (No heavy external UI libraries to ensure speed).
- **Backend:** Node.js, Express.
